- Start new Python env (python -m venv [name]) or use existing one (or using Conda)
- Install Git: https://git-scm.com/downloads if you don't have it
- Init new Git repo
- Exclude the env folder using .gitignore
- If this is your first time using GitHub on you machine: 
    - git config --global user.name "Your Name"
    - git config --global user.email "youremail@yourdomain.com"

- Create requirements.txt file: "pip3 freeze > requirements.txt"
- Publish to GitHub
- Deploy
